# common-library
Type Conversion and other common tasks easy to use

Facade to convert common types to other to common types in an easy way similar to C#:
- BigInteger
- BigDecimal
- Byte / byte
- Short / short
- Integer / int
- Long / long
- Float / float
- Double / double
- Boolean / boolean
- Character / char
- java.util.Date
- LocalDate
- LocalDateTime
- String
- Clases
